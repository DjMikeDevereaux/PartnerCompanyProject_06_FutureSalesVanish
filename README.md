## Project Overview
This project focuses on building a time series forecasting model to predict future sales of the product Vanish. It is part of the Partner Company Project within the Data Science & Business Analytics Program (EBAC).

The notebook contains the complete analytical workflow: data loading, cleaning, exploratory analysis, model selection, forecasting, and business‑oriented conclusions.

## Objectives
Analyze historical sales behavior.
Identify trends, seasonality, and cyclical patterns.
Build and evaluate a time series forecasting model (AR, ARIMA, SARIMA, etc.).
Generate clear visual predictions for upcoming months.
Provide actionable recommendations for business decision‑making.

## Tools & Technologies
Python 3.x
Pandas, NumPy
Matplotlib, Seaborn
Statsmodels (AR, ARIMA, SARIMA)
Jupyter Notebook

## Conclusions
Vanish sales show stable behavior with seasonal variations.
The chosen model provides reliable short‑term forecasts.
Monthly model updates are recommended as new data becomes available.
Incorporating external variables (marketing campaigns, pricing, seasonality) could improve accuracy.
